Kubernetes 基礎

本教程介紹了 Kubernetes 叢集編排系統的基礎知識。每個模組包含關於 Kubernetes 主要特性和概念的一些背景資訊，幷包括一個線上互動教程。這些互動教程讓你可以自己管理一個簡單的叢集及其容器化應用程式。

使用互動教程，你可以學習：

  • 在叢集上部署容器化應用程式
  • 彈性部署
  • 使用新的軟體版本，更新容器化應用程式
  • 除錯容器化應用程式

教程 Katacoda 在你的瀏覽器中執行一個虛擬終端，在瀏覽器中執行 Minikube，這是一個可在任何地方小規模本地部署的 Kubernetes 叢集。不需要安裝任何軟體或進行任何配置；每個互動性教程都直接從你的網頁瀏覽器上執行。


Kubernetes 可以為你做些什麼?

透過現代的 Web 服務，使用者希望應用程式能夠 24/7 全天候使用，開發人員希望每天可以多次釋出部署新版本的應用程式。 容器化可以幫助軟體包達成這些目標，使應用程式能夠以簡單快速的方式釋出和更新，而無需停機。Kubernetes 幫助你確保這些容器化的應用程式在你想要的時間和地點執行，並幫助應用程式找到它們需要的資源和工具。Kubernetes 是一個可用於生產的開源平臺，根據 Google 容器叢集方面積累的經驗，以及來自社群的最佳實踐而設計。
