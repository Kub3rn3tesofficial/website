他們的主要責任是保持叢集正常執行，可能需要進行週期性的維護和升級活動。<br>

**注意：** 叢集操作者不同於[操作者模式（Operator Pattern）](https://www.openshift.com/learn/topics/operators)，操作者模式是用來擴充套件 Kubernetes API 的。
