Kubernetesでサービスネットワークをモデル化するために使用されるAPIクラスのセット。

<!--more--> 
Gateway APIはKubernetesでサービスネットワークをモデル化するための拡張可能、ロール指向、プロトコル認識APIクラスのセットを提供します。
