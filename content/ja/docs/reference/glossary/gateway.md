Kubernetesでサービスネットワークをモデル化するために使用されるAPIクラスのセット。

<!--more--> 
Gateway APIは、Kubernetesにおけるサービスネットワーキングをモデル化するための、拡張性が高く、ロール指向で、プロトコル対応のAPIクラス群を提供します。
