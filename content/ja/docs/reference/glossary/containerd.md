シンプルさ、堅牢性、移植性を重視したコンテナランタイム

<!--more-->

containerd は、Linux または Windows 上でデーモンとして実行される {{< glossary_tooltip text="container" term_id="container" >}} ランタイムである。
containerdは、コンテナ・イメージの取得と保存、コンテナの実行、ネットワーク・アクセスの提供などを行うなどなど。
