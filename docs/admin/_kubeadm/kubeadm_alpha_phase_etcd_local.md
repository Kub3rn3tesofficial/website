
Generate static pod manifest file for a local, single-node etcd instance.

### Synopsis


Generate static pod manifest file for a local, single-node etcd instance.

```
kubeadm alpha phase etcd local
```

      --config string     Path to kubeadm config file (WARNING: Usage of a configuration file is experimental)
